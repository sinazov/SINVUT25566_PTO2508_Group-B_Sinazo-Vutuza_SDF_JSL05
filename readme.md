# JSL05 Project: Task Board with Local Storage

## Overview
This project enhances the previous Kanban board from JSL04 by adding **task persistence** and **creation functionality**. HTML provides the structure, with empty divs populated dynamically from an initial data array. CSS handles styling and responsive layout, while JavaScript manages interactivity, including task rendering, modal interactions, and local storage persistence. Tasks remain available after page refresh.

## Key Objectives
- Maintain **clean, modular code**. 
- Ensures **responsive design** that adapts to desktop and mobile.  
- Implement **JavaScript functionality** for adding, viewing, and storing tasks.

## Design & Responsiveness
- The "Add New Task" modal matches the Figma design and is fully responsive.  
- The task board adapts to all screen sizes.  

## Code Structure & Maintainability
- Modular JavaScript: separate responsibilities for **local storage**, **task rendering**, and **modal management**.  
- Functions are clearly named and documented for maintainability.

## Expected Outcome
A fully functional **task management system** where tasks load from local storage, can be added through a modal, and persist across sessions. The application is responsive, modular, and follows JavaScript best practices.
